# NewRoundManager2.0
SCP: Classified Site plugin.

Данный плагин заменяет существующий в игре RoundManager на новый. Отличия:
- Изменённая логика окончания раунда. В отличие от стандартного RoundManager`а, новый не заканчивает раунд, когда остаются Повстанцы Хаоса и SCP; 
- Изменённая система побед. Количество сбежавших гражданских не влияет на победу. Победа присуждается той команде, что осталась последней единственной в комплексе;
- Изменённый текст победы. Теперь он сверху и не мешает обзору.

Некоторые параметры работы плагина можно менять через конфиг (config.ini):
- **NRM_time_to_check** - время, определяющее, как часто плагин проверяет, стоит ли закончить раунд (по умолчанию: 5, то есть раз в 5 секунд);
- **restart_time** - время, которое плагин ждёт, прежде чем перезапустить раунд, после определения победы и объявления об этом всем игрокам.
